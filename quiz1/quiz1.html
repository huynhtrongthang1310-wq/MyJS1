▼ Bài 1 — Quiz nhanh: Vai trò JS, biến, toán tử

1. JavaScript trong trang web chủ yếu phụ trách gì?

HTML tạo cấu trúc, CSS tạo kiểu dáng, còn JavaScript xử lý tương tác và hành vi động của trang.

2. Từ khóa nào nên dùng cho giá trị không muốn gán lại?

const khai báo hằng số — không thể gán lại sau khi khởi tạo. Dùng let khi cần thay đổi giá trị.

3. Toán tử nào nên ưu tiên để so sánh nghiêm ngặt?

=== so sánh cả giá trị lẫn kiểu dữ liệu (strict equality). == có thể ép kiểu ngầm, dễ gây lỗi bất ngờ.
